# Prompt Effectiveness Analyzer

## Analyze and improve your AI prompts

Get instant feedback on your prompts across 5 dimensions:
- Clarity
- Specificity
- Context
- Format
- Actionability

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```
